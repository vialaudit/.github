# Vialaudit

Independent peptide-vendor audit desk. Editorial site at [vialaudit.com](https://vialaudit.com).

We publish:

- **[Vendor audits](https://vialaudit.com/vendors)** — first-hand HPLC-MS test results paired with public-source meta-analysis
- **[Peptide reference guides](https://vialaudit.com/peptides)** — research-grade summaries with citations
- **[Cross-vendor pricing index](https://vialaudit.com/pricing)** — public-catalog pricing snapshots, updated on a schedule
- **[COA archive](https://vialaudit.com/coa)** — certificate-of-analysis mirrors with extracted analytes and lot metadata

This GitHub org hosts the open-data exports of the audit desk's public datasets. The site itself is a Next.js application; nothing on this org sells, distributes, or recommends the purchase of any peptide product.

Editorial standard: numbers-first, corrections logged publicly, every vendor claim treated as a hypothesis to be tested.
